## Module 1 Quiz — Understanding AI

**1.** Which of the following best describes a Large Language Model?

- A) A database that stores and retrieves factual information
- B) Software trained on text data to predict and generate coherent language
- C) A rules-based system that follows pre-programmed instructions
- D) A real-time internet search engine with a conversational interface

**Answer:** B

---

**1b.** As of June 2026, which statement about AI agent modes is TRUE?

- A) Only ChatGPT has an agent mode that can control a browser autonomously
- B) Agent modes are still experimental and not available to regular subscribers
- C) ChatGPT, Claude, and Gemini all have agent modes that can take autonomous browser actions
- D) Agent modes require an enterprise license and are not available to individual users

**Answer:** C

---

**2.** An AI tool "hallucinating" means:

- A) The model requires more processing time than usual
- B) The model generates confident-sounding but factually incorrect content
- C) The model refuses to answer certain types of questions
- D) The model produces creative content beyond its usual capability

**Answer:** B

---

**3.** Which of the following is the MOST appropriate use of a consumer-tier AI tool (free or paid)?

- A) Analyzing a spreadsheet containing all customer account numbers
- B) Drafting a memo about a hypothetical product launch using generic details
- C) Processing employee salary data to generate a compensation report
- D) Reviewing unreleased product roadmap documents

**Answer:** B

---

**4.** The correct order of the AI maturity model from lowest to highest is:

- A) Process Automation → Team Workflow → Individual Tools → AI-Native Products
- B) Individual Tools → Team Workflow → Process Automation → AI-Native Products
- C) AI-Native Products → Process Automation → Team Workflow → Individual Tools
- D) Team Workflow → Individual Tools → AI-Native Products → Process Automation

**Answer:** B

---

**5.** Which statement about the current AI landscape is TRUE?

- A) Open-source models are always safer than closed models for enterprise use
- B) AI has replaced the need for human judgment in most knowledge work tasks
- C) AI capabilities embedded in Microsoft 365 and Google Workspace mean many organizations already have AI available without new purchases
- D) The "build" option (custom AI) is always preferable to "buy" for enterprise organizations

**Answer:** C

- A) ChatGPT (training data may be outdated)
- B) Perplexity AI (live search with citations)

**8.** Which of the following should NEVER be submitted to a consumer-tier AI tool?

- A) A draft memo using fictional company details
- B) A general question about industry trends

## Module 3 Quiz — Prompt Engineering

**9.** In the RACE prompting framework, what does the "C" stand for?

- A) Content
- B) Context
- C) Clarity
- D) Constraints

**Answer:** B

---

**10.** Which prompt is most likely to produce a high-quality executive output on the first attempt?

- A) "Write something about our sales strategy."
- B) "Help me with sales."
- C) "You are a VP of Sales at a B2B SaaS company. Write a 300-word internal memo to the sales team outlining Q4 priorities. Tone: direct and motivating. Focus on: pipeline hygiene, enterprise deals, and Q4 close rate. Bullet format."
- D) "Write a memo about Q4 sales goals in bullet points."

**Answer:** C

---

**11.** When iterating on an AI output, which approach is MOST effective?

- A) Always start a new conversation — previous context slows the model down
- B) Continue the conversation with specific refinement instructions ("Make it shorter," "Add risks section")
- C) Delete the output and try a completely different topic
- D) Ask the AI to rate its own output and suggest improvements

**Answer:** B

- A) January 1, 2026

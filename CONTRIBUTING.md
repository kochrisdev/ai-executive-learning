# Contributing

This is a living curriculum. Contributions that keep it accurate, current, and useful are welcome.

---

## What to Contribute

**High-value contributions:**
- Corrections when a tool changes or a fact is outdated
- New prompt patterns that work well for executives
- Exercise improvements based on running the course with real participants
- New templates that fill gaps in the existing set
- Facilitator observations that improve delivery

**Out of scope:**
- Theoretical content without practical application
- Marketing or promotional content
- Content requiring a specific paid tool not already in the curriculum

---

## How to Contribute

1. **For small corrections** (typo, outdated price, changed UI): Open an issue with the [Curriculum Feedback](.github/ISSUE_TEMPLATE/curriculum-feedback.md) template or submit a PR directly.

2. **For content additions**: Open an issue first to discuss before writing. This avoids duplicate work.

3. **For tool updates**: Use the [Tool Update](.github/ISSUE_TEMPLATE/tool-update.md) issue template.

---

## PR Guidelines

- Keep PRs focused — one topic per PR
- Test any prompts or exercises you add before submitting
- Match the existing tone and style (direct, practical, no unnecessary jargon)
- Do not add tools without justifying why they belong in an executive curriculum
- Do not commit participant data, real company names, or personal information

---

## Review Process

PRs are reviewed within 7 days. Reviews focus on:
- Accuracy: is this factually correct?
- Relevance: does this serve the executive audience?
- Consistency: does this match the tone and structure of the curriculum?
- Safety: does this introduce any data privacy or security concerns?

---

## Code of Conduct

This project follows standard open-source community norms. Respectful, constructive contributions only.

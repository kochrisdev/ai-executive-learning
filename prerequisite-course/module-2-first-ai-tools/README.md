## Before You Start This Module

You must have accounts created at:
- [ChatGPT Plus](https://chat.openai.com) — $20/month (required for GPT-4o)
- [Claude Pro](https://claude.ai) — $20/month (required for extended context)
- [Perplexity](https://perplexity.ai) — free tier sufficient; Pro optional

> **Data Privacy Note:** Do not submit any confidential company information, customer data, or personal data into these consumer-tier tools until your organization has reviewed its AI usage policy.

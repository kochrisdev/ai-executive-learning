# Lesson 3.4 — Iteration and Refinement

**Time:** ~10 minutes

---

## Iteration Is Expected — Not a Sign of Failure

Getting a perfect output on the first prompt is the exception, not the rule. Expert AI users iterate rapidly. The goal is to get from blank page to polished output in 2–3 exchanges, not 1.

---

## Refinement Commands That Work

Use these follow-up instructions to improve any output:

### Adjust Length
- "Make this 30% shorter without losing the key points."
- "Expand the risks section with 3 more specific examples."
- "Give me a 1-sentence version of this for a subject line."

### Adjust Tone
- "Rewrite this in a more direct, less diplomatic tone."
- "Make this warmer and more human — less corporate."
- "Elevate the language for a board-level audience."

### Adjust Structure
- "Reorganize this as a table instead of paragraphs."
- "Move the recommendation to the top — lead with the answer."
- "Split this into a two-part structure: context first, then action."

### Add or Remove Content
- "Add a section on implementation timeline."
- "Remove all references to cost — that will be covered separately."
- "Include a counterargument and rebuttal to the main recommendation."

### Request Alternatives
- "Give me 3 alternative versions of the opening paragraph."
- "Rewrite this assuming the audience is skeptical, not supportive."
- "Write a more conservative version and a more aggressive version."

---

## Multi-Turn vs. Starting Fresh

**Continue the conversation when:**
- You are refining a specific output
- The AI has context it needs to keep (company details, document it analyzed)
- You are iterating toward a final version

**Start fresh when:**
- You want a completely different angle, not a refinement
- The current conversation has gone off-track
- You are switching to a different task entirely

---

## The "Why" Technique

If output quality is consistently poor for a task, add this to the end of your prompt:
```
Before answering, tell me what additional information would help you
give me a better response.
```

The AI will tell you exactly what context is missing. Provide it, then re-run the prompt.

---

## Time Benchmark

Experienced AI users move from blank page to polished first draft in under 5 minutes for most executive tasks. The iteration loop is:
- Prompt → 30 seconds for output
- Review → 1 minute
- Refine → 15 seconds to type follow-up
- Final output → 30 seconds

Practice until this rhythm feels natural.

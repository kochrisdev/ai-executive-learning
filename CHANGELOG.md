## [2.0.0] — June 2026

### Major Update: 2026 AI Landscape Refresh

This release updates all curriculum content to reflect the significant AI developments of 2025–2026.

---

### New Capabilities Added

#### Models
- **Claude Opus 4.8** (May 2026): Documented new agentic capabilities, 84% web automation benchmark, 2.5× Fast Mode speed, 4× better error acknowledgment, 1M token context window
- **Claude Max tier** ($100/month): Added tier comparison table
- **GPT o3 reasoning model**: Added to ChatGPT tool guide with use-case guidance
- **Gemini 2.5 Pro + Gemini Spark**: Added Google's new daily brief and active agent features
- **Agent Mode**: All three major models now have browser-control agent modes — documented in Module 2 (ChatGPT, Claude), Phase 4, and governance policy

#### Video AI (Major Change)
- **Removed:** Sora (OpenAI) — discontinued March 2026
- **Added:** Veo 3.1 (Google) — now recommended as primary video tool; native audio, 4K 60fps
- **Added:** Runway Gen-4.5 — recommended for professional creative workflows
- **Added:** Kling 3.0 — recommended for high-volume/value use cases
- **Added:** Seedance 2.0 — emerging tool with strong image-to-video performance
- Updated Phase 2 tool table with status indicators

#### Automation
- **n8n** added as a major Make.com alternative with detailed comparison
- n8n positioned as the recommended option for regulated industries requiring self-hosted deployment
- Updated Phase 3 tool table and decision framework

#### Knowledge Management
- **Google NotebookLM** added as a dedicated executive tool section in Lesson 1.2
- NotebookLM Enterprise tier documented (VPC-SC compliant, audit trails, no training on data)
- Added to Phase 3 as a knowledge management tool

#### Microsoft 365 Copilot (2026 Updates)
- Agent mode in Word, Excel, PowerPoint documented
- Claude Opus 4.8 now available within M365 Copilot (May 2026)
- Customizable meeting recap templates documented
- Video meeting recaps feature added

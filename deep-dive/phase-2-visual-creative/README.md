# Phase 2: Visual & Creative AI

**Day:** Day 1, Afternoon
**Duration:** 3 hours
**Facilitator:** Lead instructor
**Room setup:** Same as Phase 1

---

## Session Objectives

By the end of this phase, participants will:
- Understand the landscape of visual and creative AI tools
- Have generated an AI image or visual asset for a real communication
- Know how to brief AI video tools without prior creative experience
- Have explored AI audio/voice tools relevant to their work

## Exercise: Real Creative Output (30 min)

Each participant chooses one creative task relevant to their work and completes it using AI tools:

**Options:**
- Generate a header image for an upcoming presentation or internal communication
- Create a short (30–60 second) concept video brief using Runway
- Produce a background music track for a video using Suno
- Use Synthesia to create a sample AI presenter video

**Prompt template for image generation:**
```
A professional, photorealistic image for [context/purpose].
Style: [corporate / creative / minimalist / bold]
Color palette: [your brand colors or preference]
No text in the image.
Aspect ratio: [16:9 for presentations / 1:1 for social / 9:16 for vertical]
```

---

## Facilitator Notes

- Many executives are surprised by the quality of image generation — let the outputs speak for themselves before commenting
- Runway requires a free account signup — have participants do this during the Phase 1 break if possible
- Wispr Flow demo is highly relatable: speak a prompt naturally instead of typing — executives who hate typing love this
- For the gallery share, display outputs on the main screen — peer reaction to quality is a strong motivator
- Some participants will be uncomfortable with creative tools ("that's not my job") — redirect to the use case: briefing agencies, reviewing AI outputs, and making faster decisions requires fluency with these tools

# Assessment Rubrics

*Use these to evaluate participant outputs from both the Prerequisite Course exercises and the Deep Dive.*

---

## Prerequisite Course Exercise Rubric

Applies to all 5 module exercises submitted before the Deep Dive.

| Criterion | 3 — Excellent | 2 — Adequate | 1 — Needs Work |
|---|---|---|---|
| **Completion** | All sections fully completed | Most sections completed with substance | Incomplete or cursory responses |
| **Specificity** | Real, specific examples from actual work | Some specifics; some generic | Vague or hypothetical throughout |
| **Reflection quality** | Demonstrates genuine thinking about AI in their context | Surface-level reflection | Little to no reflection |
| **AI engagement** | Evidence of actually using the tools (exercises require live use) | Partial tool use | Appears completed without using tools |

**Minimum to attend Deep Dive:** Score of 2 or higher on all four criteria across all 5 exercises.

---

## Real Task Sprint (Phase 1) Rubric

Evaluates the output produced during the Day 1 hands-on exercise.

| Criterion | 3 | 2 | 1 |
|---|---|---|---|
| **Prompt quality** | Full RACE structure; specific, contextual | Partial RACE; missing context or format | Vague prompt with minimal structure |
| **Output usability** | Output could be used (with light editing) in real work | Output is a reasonable starting point needing significant editing | Output is generic or not fit for purpose |
| **Iteration** | At least one meaningful refinement made | Minimal iteration | Accepted first output without refinement |
| **Business relevance** | Task directly related to real work | Loosely related | Generic or test task |

---

## Custom GPT (Phase 4) Rubric

Evaluates the custom AI agent built during Phase 4.

| Criterion | 3 | 2 | 1 |
|---|---|---|---|
| **Use case clarity** | Specific, well-scoped use case with clear audience | Reasonable use case; somewhat broad | Vague use case or not clearly defined |
| **System prompt quality** | Clear role, behavior rules, tone, and format specified | Partial specification; some elements missing | Generic or minimal system prompt |
| **Knowledge files** | Relevant document(s) uploaded that meaningfully shape responses | Document uploaded but not well-tailored | No knowledge files or irrelevant files |
| **Demonstrated function** | Agent produces useful output on real test queries | Agent works but outputs are generic | Agent does not produce useful output |

---

## 90-Day Plan (Phase 5) Rubric

Evaluates the plan presented at the close of the Deep Dive.

| Criterion | 3 | 2 | 1 |
|---|---|---|---|
| **North Star** | Specific, measurable, time-bound | Directionally correct but vague | Missing or unmeasurable |
| **Tool selection** | Specific tools chosen with clear rationale | Tools named but rationale unclear | No specific tools identified |
| **Milestones** | Concrete milestones with dates and owners | Milestones present but vague | No milestones or all "TBD" |
| **Metrics** | Specific measurable outcomes with baselines | Some metrics; no baselines | No metrics defined |
| **Risk awareness** | Real risks identified with mitigations | Some risks acknowledged | No risks considered |
| **Commitment** | Clear personal commitment expressed | Willing but hedged | Non-committal |

**Completion standard:** Score of 2 or higher on all six criteria = plan is ready to implement.

## Post-Program Assessment (30 and 90 Days)

Collect these via survey at 30 and 90 days post-Deep Dive:

| Question | Response Format |
|---|---|
| Are you using AI tools at least 3x per week? | Y / N |
| How many hours per week are you saving with AI? | Number |
| How many workflows have you automated? | Number |
| Is your custom GPT still in use? | Y / N |
| Has your team adopted AI tools? | % of team using weekly |
| Did you present your 90-day plan to leadership? | Y / N |
| What is your current AI maturity level (1–4)? | Number |
| NPS: How likely are you to recommend this program? | 0–10 |
| One thing that would make this program better: | Open text |
